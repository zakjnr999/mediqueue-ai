# MediQueue AI

MediQueue AI is an AWS-powered digital patient check-in and hospital queue management system.

## Core MVP Flow

Patient check-in → symptom submission → Amazon Bedrock analysis → staff review → queue assignment → status updates.

## Planned AWS Services

- Amazon Bedrock
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon CloudWatch
- AWS Amplify
- Amazon Cognito
